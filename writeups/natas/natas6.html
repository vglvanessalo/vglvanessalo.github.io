NATAS6

18/05/2019

OBJECTIVE:

The aim of this level is to find the password for the natas6 site.

TOOLS:

Burp Suite

METHOD:

Using the password obtained from level 4-5, login to the natas5 site. The following page (see below) once successfully logged in will display a message saying that access is disallowed because you are not logged in.



Since we are not logged in, we need to find a way to login without using a password. In order to do that, we need to edit a cookie that is stored upon visiting the site. In order to edit the cookie, we need to use Burp Suite. Once Burp Suite is installed, follow the instructions on the Burp Suite website to setup a proxy connection and to ensure that your browser is connected to the Burp Suite listener.

Once Burp Suite is configured, go to the natas5 site and open up Burp Suite. Here we go to the Proxy tab and then the Intercept panel (ensuring intercept is on) refresh the natas5 page and the cookie information will appear in Burp Suite (see below).



Right click on any part of the cookie information and then click on "Send to Repeater".



Click on the Repeater tab and then click on the Params tab. Under the Params tab, you will see a list of cookies, notice that there is a "loggedin" cookie which shows a value of "0" meaning that you are not logged in. In order to edit the cookie so they we are logged in, we need to change the value of "loggedin" from "0" to "1". Once the value is changed, press the "Go" button in the top left corner of the window.



Once the page has reloaded with the edited cookie, the response section on the right will show the source code of the reloaded page. The new message will show "Access granted." and the password for natas6 is given (see below).
