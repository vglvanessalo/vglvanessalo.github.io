NATAS8

18/05/2019

OBJECTIVE:

The aim of this level is to find the password for the natas8 site.

TOOLS:

No tools are needed for this level.

METHOD:

Using the password obtained from level 6-7, login to the natas7 site. The following page (see below) once successfully logged in will display a Home and About link.



As with any website when looking for information, the first step is to look at the source code. To do so, right click in your browser window and select "view page source". The source code will then be displayed in a new tab (if using chrome).

Look through the source code, paying particular attention to any comments in the code. In a comment, there is a hint as to where the password can be found by following the "/etc/natas_webpass/natas8" directory (see below).



To access the "/etc/natas_webpass/natas8" page from the natas7 site, add "/etc/natas_webpass/natas8" to the end of the URL in the browser address bar and the page should load to reveal the password for natas8 (see below).
