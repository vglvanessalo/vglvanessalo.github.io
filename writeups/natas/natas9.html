NATAS9

18/05/2019

OBJECTIVE:

The aim of this level is to find the password for the natas9 site.

TOOLS:

CyberChef

METHOD:

Using the password obtained from level 7-8, login to the natas8 site. The following page (see below) once successfully logged in will display an input field asking for a secret and a link to view the source code.



Click on the "View sourcecode" link. In this particular source code, there is a encoded secret (see below).



Also take notice that the secret is printed as a hexidecimal, in reverse and is encoded with base 64 (see below).



In order to decode the secret, we need to use CyberChef. Firstly, converting the hexidecimal back into ASCII (see below).



Then we need to reverse the the ASCII string (see below).



Then decode the string from base 64, giving us the secret (see below).



Then we need to go back to the natas8 site and input the secret we obtained from CyberChef and then click submit (see below).



The page will reload displaying the password for natas9 (see below).

The password for natas9 is: W0mMhUcRRnG8dcghE4qvk3JA9lGt8nDl
