NATAS5

17/05/2019

OBJECTIVE:

The aim of this level is to find the password for the natas5 site.

TOOLS:

ModHeader (browser extension)

METHOD:

Using the password obtained from level 3-4, login to the natas4 site. The following page (see below) once successfully logged in will display a message saying that you are visiting from an unauthorised page and that authorised users should only come from natas5.



Since the we are accessing the site from natas4, we need to spoof the request header so that it refers to natas5. To do this we need to install an extension called ModHeader, which allows use to refer the request header to natas5.

Once ModHeader is installed, go to the natas4 site and open up the extension. Here we will add "Referer" into the name field (ensuring the row is checked), and the URL to the natas5 site into the value field (see below).



Once all the fields have been filled in refresh the page. The page should refer to natas5 giving authorised access to the natas4 page and the password should appear(see below).
