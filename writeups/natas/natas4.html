NATAS4

17/05/2019

OBJECTIVE:

The aim of this level is to find the password for the natas4 site.

TOOLS:

No tools are needed for this level.

METHOD:

Using the password obtained from level 2-3, login to the natas3 site. The following page (see below) once successfully logged in will display a message saying that there is nothing on the page.



As with any website when looking for information, the first step is to look at the source code. To do so, right click in your browser window and select "view page source". The source code will then be displayed in a new tab (if using chrome).

Look through the source code, paying particular attention to any comments in the code. In the source code, there is a comment which says "No more information leaks!! Not even Google will find it this time..." (see below).



Given the hint that "Not even Google will find it" points to the idea of web crawlers, spiders and internet bots. Web crawlers, spiders and internet bots are often used by search engines and sites to update web content and often access these sites without approval, these bots also take requests from site that include a "robots.txt" file.

To access the "robots.txt" file from the natas3 site, add "/robots.txt" to the end of the URL in the browser's address bar. The "robots.txt" file should load showing a "/s3cr3t/" directory (see below).



To access the "/s3cr3t/" directory from the natas3 site, add "/s3cr3t/" to the end of the URL in the browser's address bar. The directory should load showing a folder containing a .txt file (see below).



Click on the "user.txt" file and the password for natas4 appears in the file (see below).
