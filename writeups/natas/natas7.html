NATAS7

18/05/2019

OBJECTIVE:

The aim of this level is to find the password for the natas7 site.

TOOLS:

No tools are needed for this level.

METHOD:

Using the password obtained from level 5-6, login to the natas6 site. The following page (see below) once successfully logged in will display an input field asking for a secret and a link to view the source code.



Click on the "View sourcecode" link. In this particular source code, there is a directory to a file called "secret.inc" in the script which is accessible through "includes/secret.inc" (see below).



To access the "secret.inc" file from the natas6 site, add "includes/secret.inc" to the end of the URL in the browser address bar and the "secret.inc" file should load (see below). The secret will appear in the file.



Go back to the natas6 site and input the secret that was found in the "secret.inc" file. Then press submit (see below).



The page will reload with a message displaying the password for natas7 (see below).
